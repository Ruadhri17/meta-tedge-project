## Getting started

1. Install kas

    ```
    pip install kas
    ```

2. Checkout the repo

    ```sh
    git clone https://github.com/reubenmiller/meta-tedge-project.git
    ```

3. Build the project

    **Raspberry Pi**

    ```sh
    python3 -m kas build ./projects/demo.yaml core-image-tedge-mender
    ```
